Vector Spaces

The idea of a vector space is a generalization of the concepts we’ve studied so far to other sets of objects.

Let V be a set of objects (vectors) with addition (+) defined. V is a vector space if

  1. 1.

    V is closed under addition.

    If u→∈V and w→∈V, then u+w→∈V. In other words, V is closed under vector addition.

  2. 2.

    V is closed under scalar multiplication.

    If c is a scalar in ℝ or ℂ, and u→∈V, then c⁢u→∈V.

  3. 3.

    Zero element.

    There exists some ‘zero element’ z→∈V such that u→+z→=u→ for all u→∈V.

  4. 4.

    Additive inverses.

    For every u→∈V, there exists some m→∈V such that u→+m→=m→+u→=z→.

These concepts and rules apply to a great number of spaces, such as the familiar real numbers ℝ and vectors ℝn, but they also apply to sets such as polynomials, continuous functions, and matrices.

Not everything is a vector space – even small differences can matter. Consider the set

S={a⁢(x)|a⁢(x) has degree 3}

Consider for the polynomials 3⁢x3 and 2⁢x3-7. Both are of degree 3 and their sum 5⁢x3-7 is also of degree 3. On the other hand, the polynomials 3⁢x3 and -3⁢x3 are also both of degree 3, and their sum is the zero polynomial, which is of degree 0. Thus the set is not a vector space.

Consider next the matrix equation

𝐀⁢x→ =0→

Call the solution set S by

S={x→0∈ℝn:𝐀⁢x→0=0→}

Does this space have the properties of a vector space? That is, do the solutions to the above matrix equation form a vector space? We will soon see that it does.

Let u→,v→∈S and take c to be a scalar.

  1. 1.

    Additive closure:

    Observe that

    A⁢(u→+v→)=A⁢u→+A⁢v→=0→+0→=0→

    Thus u→+v→∈S.

  2. 2.

    Scalar multiplication:

    A⁢(c⁢u→)=c⁢A⁢u→=c⁢0→=0→

    Thus c⁢u→∈S.

  3. 3.

    Zero element:

    If x→0=0→, then 𝐀⁢x→0=0→ for any 𝐀. 0→+u→=u→, thus S will always have a zero element.

  4. 4.

    If 𝐀⁢x→0=0→, then -𝐀⁢x→0=-0→=0→ and so 𝐀⁢(-x→0)=0→.

Operations and Properties of Vector Spaces

There are several additional important properties of vector spaces that are important to the structure of vector spaces.

  1. 1.

    V is closed under +: If u→∈V and w→∈V, then u+w→∈V. In other words, V is closed under vector addition.

  2. 2.

    V is closed under scalar multiplication: If c is a scalar in ℝ or ℂ, and u→∈V, then c⁢u→∈V.

  3. 3.

    Zero element: There exists some z→∈V such that u→+z→=u→ for all u→∈V.

  4. 4.

    Additive inverses: For every u→∈V, there exists some m→∈V such that u→+m→=m→+u→=z→.

  5. 5.

    Commutativity of addition: u→+v→=v→+u→ for all u→,v→∈V.

  6. 6.

    Associativity: (u→+v→)+w→=u→+(v→+w→)

  7. 7.

    For scalars c,d∈ℝ, then c⁢(d⁢u→)=(c⁢d)⁢u→

  8. 8.

    Distributivity: c⁢(u→+w→)=c⁢u→+c⁢w→

  9. 9.

    Distributivity: (c+d)⁢u→=c⁢u→+d⁢u→

Subspaces

Let S⊆V such that

  1. 1.

    S is non-empty.

  2. 2.

    S is a vector space in its own right with the operations of V.

Given a properly selected subset of a vector space, many of the properties of the vector space will be carried over to the subspace. Many such properties carry over because they are results of the operations being the same between both sets.

Let us look at some examples of subspaces. Without naming a particular vector space, we can say some things already:

Question:

Is {0→}⊆V a subspace? Yes.

Question:

Is V a subspace? Yes.

Let us consider a more complicated example. Consider ℝ2. Is the x-axis a vector space? Call this set S:

S={(x0):x∈ℝ}

Then if u→,v→∈S,

u→=(a0) v→=(b0)

Then

u→+v→=(a+b0)∈S c⁢u→=(c⁢a0)∈S

So S is a subspace of ℝ2. Similarly, the y-axis is also a subspace by similar arguments.

Consider the following sets:

T ={(xy):y=-5⁢c} W ={(xy):y=x+1}

are they vector spaces?

The set T is simply a line. It has slope -5 and passes through the origin. Let u→,w→∈T. Then for some a,b∈ℝ,

u→=(a-5⁢a) w→=(b-5⁢b)

Then

u→+w→ =(a+b-5⁢(a+b))∈T

Thus T is closed under addition. Furthermore,

c⁢u→=c⁢(a-5⁢a)=(a⁢c-5⁢a⁢c)∈T

so T is closed under scalar multiplication.

Next consider W. It is also a line, but we can note immediately that it does not contain the zero vector. What if we append a zero element to W? That is, let F=W∪{0→}. We find that there are still problems – consider for instance the two vectors

u→=[12] w→=[23]

both are clearly members of W, but their sum is not:

u→+w→=[35]

Since 3+1=4≠5, thus u→+w→∉F. Thus W is not a vector space, and so it is not a subspace of ℝ2.

Question:

What are all the subspaces of ℝ2?

Guess: ℝ2,{0→}, and any line passing through (00).

Question:

What about ℝ3,ℝ4, and beyond?

A good place to start is with linear objects. What kind of linear objects exist in ℝ3? We know of lines and planes.

Consider

S={(xyz):x+y+z=0}

Is S a subspace of ℝ3. If we observe that S is the set of solutions to a homogeneous system, then it must be, from our earlier observations.

Next consider that the system can be rewritten in terms of a dot product:

x+y+z =0
⟨x,y,z⟩⋅⟨1,1,1⟩ =0

Thus S is equivalently

S={w→∈ℝ3:w→⋅⟨1,1,1⟩=0}

in other words, S is the set of vectors orthogonal to ⟨1,1,1⟩.

Note that

  1. 1.

    0→∈S

  2. 2.

    Let u→1,u→2∈S. Then

    (u→1+u→2)⋅⟨1,1,1⟩=u→1⋅⟨1,1,1⟩+u→2⋅⟨1,1,1⟩=0+0=0
  3. 3.
    (c⁢u→1)⋅⟨1,1,1⟩ =c⁢(u→⋅⟨1,1,1⟩)=c⁢(0)=0

The vector ⟨1,1,1⟩ was chosen arbitrarily, and in fact any vector would induce a vector space composed of vectors orthogonal to it. Without proof, we seem to be finding that the subspaces of ℝ3 are the zero subspace, lines, planes, and the whole of ℝ3 itself.

Consider again the set of polynomials of degree 3 or less:

ℙ3⁢(x)={a⁢(x):a⁢(x) has degree ≤3}

Consider the subset S⊆ℙ3⁢(x),

S={a⁢(x):a⁢(0)=0}

This set contains elements such as x,-x3-7⁢x2+π⁢x,2⁢x2,0.

Is S a subspace of ℙ3⁢(x)? Let a⁢(x),b⁢(x)∈S. Then a⁢(0)=b⁢(0)=0 and hence

a⁢(x)+b⁢(x) =c⁢(x)
a⁢(0)+b⁢(0) =c⁢(0)
0+0 =c⁢(0)

Consider that we can identify a polynomial of degree 3 or less with a vector of length 4.

a⁢x3+b⁢x2+c⁢x+d⟷(abcd)

The set S can similarly be represented:

a⁢x3+b⁢x2+c⁢x+0⟷(abc0)

The two spaces ℙ3⁢(x) and ℝ4 seem to behave very similarly. In fact, there is a term called isomorphism which denotes that their structures are indeed essentially identical. The point of studying this is that these two seemingly different spaces behave in the same way. It is thus beneficial to study a generic model rather only sticking to one space, so that the concepts learned about vector spaces can be more widely applied.
